Odpowiedzi na pytania teoretyczne z JavaScript

1) Kiedy powstał JavaScript (podaj datę)?

JavaScript został stworzony w 1995 roku przez Brendana Eicha, pracującego w firmie Netscape.

2) Dlaczego jest to język skryptowy?

JavaScript jest językiem skryptowym, ponieważ jego kod jest interpretowany bezpośrednio przez przeglądarkę podczas wykonywania. Nie wymaga kompilacji przed uruchomieniem.

3) Gdzie jest wykonywany JS (klient lub serwer)?

JavaScript jest głównie wykonywany po stronie klienta (w przeglądarce), ale za pomocą technologii takich jak Node.js może być również uruchamiany po stronie serwera.

4) Dlaczego JavaScript to bezpieczne narzędzie?

JavaScript jest uważany za bezpieczne narzędzie, ponieważ działa w tzw. "sandboxie", co oznacza, że nie ma bezpośredniego dostępu do plików systemu operacyjnego użytkownika ani innych zasobów urządzenia, poza tymi, na które użytkownik wyraził zgodę.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?

Tak, JavaScript jest językiem obiektowym, ponieważ wspiera tworzenie obiektów, które mogą mieć właściwości i metody. Jego model programowania opiera się na prototypach, co pozwala na dziedziczenie właściwości i metod między obiektami.

6) Wymień zalety JS:

  • Zaleta 1: JavaScript działa na wszystkich nowoczesnych przeglądarkach bez potrzeby instalowania dodatkowego oprogramowania.
  • Zaleta 2: Umożliwia tworzenie interaktywnych stron internetowych, które reagują na akcje użytkownika w czasie rzeczywistym.
  • Zaleta 3: Dzięki Node.js JavaScript może być używany do tworzenia zarówno front-endu, jak i back-endu aplikacji.

7) Framework to:

Framework to zbiór narzędzi i bibliotek, które ułatwiają tworzenie aplikacji poprzez dostarczanie gotowych struktur i komponentów. Przykładami frameworków JavaScript są React, Angular i Vue.js.

8) Do aplikacji mobilnych można użyć Framework:

Tak, do tworzenia aplikacji mobilnych można używać frameworków takich jak React Native, który pozwala pisać aplikacje mobilne przy użyciu JavaScript.

9) Aplikacje responsywne to:

Aplikacje responsywne to takie, które automatycznie dostosowują swój wygląd i układ do rozmiaru ekranu urządzenia, na którym są wyświetlane (np. komputer, tablet, telefon komórkowy).

10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu:

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Ten fragment kodu tworzy przycisk, który po kliknięciu otwiera nową stronę w nowym oknie przeglądarki za pomocą funkcji `WinOpen`.

11) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu:

window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Ten fragment kodu otwiera nowe okno przeglądarki, w którym wyświetlana jest strona "obraz.html". Okno ma ustawioną wysokość 280 pikseli i szerokość 160 pikseli, a także jest pozbawione pasków narzędzi (toolbar), katalogów (directories) i menu (menubar). Okno jest wyświetlane na pozycji 200 pikseli od góry i 200 pikseli od lewej strony ekranu.

12) Opisz następującą instrukcję window.close():

Instrukcja `window.close()` służy do zamknięcia bieżącego okna przeglądarki. Zwykle jest używana w oknach otwartych za pomocą `window.open()`.
